## Tech Stack

1. HTML
2. CSS
3. JavaScript
4. SCSS
5. Glide.js - For the carousels and slides
6. Three.js - For the virtual environment
7. Axios.js - For fetching data from API
8. Chart.js - For data visualisation
9. RAWG API - For Video Game Database
